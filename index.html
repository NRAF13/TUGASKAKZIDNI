ABOUT FIVEM

FIVEM is mod of GTA 5 online to play with dedicated server and give freedom to developer to build the server wathever they likes Any gamemodes is available like DEATHMATCH, DRIFT, ROLEPLAY, FREEROAM, ZOMBIE and many more.
ROLEPLAY is one of popular gamemode, lot of server is use ROLEPLAY gamemode, we can take nay role in this gamemode like POLICE, EMS, MAYOR and many more. FIVEM give permission to server to use mod as feature, like add-on cars, building, skin, outfit and many more.

PROGRAMMING LANGUAGE

FIVEM using Lua programming language as main language for server. FIVEM also use html and css to create menu in screen player. C# will use cor to build a custom launcher to connect the server


LIMITATION

FIVEM have limit for maximum player, because is use GTA Online network, but now FIVEM has onesync.
WHAT IS ONESYNC?? onesync is FiveM feature to allow dedicated server has above 64 players slot without any problems.
